# 要件定義書 - Saisoku-AI (催促AI)

AIが煽りながらタスク管理を促す、自己研鑽に特化したタスク管理アプリ。

## 1. コンセプト
「昨日立てた目標、まさか忘れてないよね？」
AIが少し生意気な口調で進捗を催促し、証拠画像がない限りタスク完了を認めない、自分に甘えられない環境を提供します。

## 2. ターゲットユーザー
- 自己研鑽を始めたいが、三日坊主になりがちな人
- AIに厳しく（あるいは煽られて）管理されたい人
- 成果を視覚的（エビデンス画像）に残したい人

## 3. MVP主要機能
### 3.1 タスク（ノルマ）管理
- **前日予約:** 翌日のノルマを前日までに設定する。
- **エビデンス付き完了報告:** タスク完了時には必ず「証拠画像」のアップロードを必須とする。

### 3.2 煽りAI催促 & 報酬 (LINE連携)
- **スケジュール:**
    - **平日:** 19:00にリマインド & 進捗チェック（煽り）。
    - **休日:** 朝9:00に一次チェック（煽り）、深夜24:00に最終チェック。
- **煽りメッセージ:** 「今日は何か成長しましたか？」「昨日言ったこと、もう忘れちゃった？」といった刺激的なメッセージをAI（MOTIA経由）が生成。
- **完了報酬:** タスク完了報告時には「今日も成長できましたね！」「素晴らしい成果です」といった承認・報酬メッセージをAIが提供。

### 3.3 エビデンス管理
- アップロードされた画像は履歴として閲覧可能（自分の頑張りを可視化）。

## 4. 技術スタック
- **Frontend:** Next.js (App Router)
- **Backend/Workflow:** MOTIA
- **Database/Storage:** Supabase (PostgreSQL / Storage)
- **AI:** OpenAI API (GPT-4o 等)
- **Messaging:** LINE Messaging API

## 5. ユーザーフロー (MVP)
1. **[前日夜]** Next.jsアプリで翌日のノルマを入力。
2. **[当日 指定時刻]** MOTIAがスケジュール実行され、LINEで煽りリマインドが届く。
3. **[当日中]** ユーザーがタスク実行。
4. **[タスク完了時]** アプリを開き、エビデンス画像をアップロードして「完了」ボタンを押す。 -> AIから「成長」を称える報酬メッセージが届く。
5. **[当日最終チェック時]**
   - 未完了の場合: AIから強烈な煽りLINEが届く。

## 6. 非機能要件
- PC/デスクトップでの利用を想定したモダンなWebデザイン。
- LINE通知のリアルタイム性。
